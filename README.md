# eflyer website

This is a responsive multipage e-commerce website built using HTML, CSS, JavaScript and Bootstrap.

## Features

- Responsive design
- Multiple pages
- Product listing page
- Product detail page
- Shopping cart page
- Checkout page
- Contact us page

## Getting Started

1. Clone the repository.
2. Open the `index.html` file in your browser.
